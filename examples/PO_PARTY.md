## Advanced Usage

### Programmatic API

```python
from po_core.party_machine import PhilosopherPartyMachine, PartyMood

# Create party machine
party = PhilosopherPartyMachine()

# Suggest configuration
config = party.suggest_party(
    theme="ethics",
    mood=PartyMood.CHAOTIC,
    custom_prompt="How should we approach AI ethics?"
)

# Display configuration
print(f"Party size: {config.expected_size}")
print(f"Philosophers: {', '.join(config.philosophers)}")
print(f"Expected emergence: {config.expected_emergence:.1%}")

# Run the party with Po_self
from po_core.po_self import PoSelf

po = PoSelf(
    philosophers=config.philosophers,
    enable_ethics_guardian=True,
)

result = po.generate("How should we approach AI ethics?")
print(result.text)
print(f"Actual emergence: {result.metrics['semantic_delta']:.3f}")
```

### Quick Factory Function

```python
from po_core.party_machine import create_party

# One-line party creation
config = create_party("existence", "balanced")
print(f"Ready to party with: {', '.join(config.philosophers)}")
```

## FAQ

**Q: Is this just random philosopher selection?**
A: No! Every suggestion is based on empirical analysis of 10,600 reasoning sessions. The combinations are optimized for emergence, tension, and insight quality.

**Q: Can I use specific philosophers I want?**
A: Yes! Use the programmatic API and create a custom PartyConfig, or use Po_self directly with your chosen philosophers.

**Q: Does it actually run reasoning, or just simulate?**
A: Po_Party runs **real** Po_self reasoning with the selected philosophers. The metrics you see are genuine outputs from the philosophical ensemble.

**Q: What if I get boring results?**
A: Try increasing the mood to "chaotic" or "critical". Higher tension = more interesting dynamics. Also, make sure your question is open-ended!

**Q: Can I save party configurations?**
A: Currently not implemented, but you can save the philosopher list from the console output and reuse it with Po_self directly.
