# ビューア / Viewer

推論過程の可視化とユーザーインターフェースに関する設計ドキュメントを格納しています。

## 📋 概要

Po_core Viewerは、複雑な哲学的推論プロセスを視覚的に表現し、ユーザーが直感的に理解できるインターフェースを提供します。

## 🎯 主要機能

### 基本ビューア（v0.3）
- 推論チェーンの表示
- ステップごとのナビゲーション
- 基本的なインタラクション

### 進化版ビューア（v2）
- 高度な可視化
- リアルタイム更新
- カスタマイズ可能なビュー

### 特殊可視化
- **倫理ゆらぎ可視化**: 倫理的ジレンマの視覚表現
- **圧力分析**: freedom_pressure_tensorの可視化
- **自己語り構造**: Po_selfの構造表示

## 🎨 可視化の種類

### グラフ表示
- 推論チェーンのフローグラフ
- 哲学者間の影響関係
- テンソル空間の投影

### ヒートマップ
- 圧力場の分布
- 哲学的重み付け
- 信頼度の可視化

### タイムライン
- 推論過程の時系列表示
- イベントの発生順序
- 処理時間の可視化

### 3D表現
- 多次元テンソル空間
- 哲学的空間の立体表示
- インタラクティブな探索

```
推論実行
    ↓

    ↓

    ↓

```

## 🌟 独自の特徴

### リアルタイム更新
推論が進行するにつれて、リアルタイムで表示を更新します。

### 多層ビュー
同じ推論を異なる抽象度で同時に表示可能です。

### インタラクティブ探索
ユーザーが推論の各ステップを自由に探索できます。
