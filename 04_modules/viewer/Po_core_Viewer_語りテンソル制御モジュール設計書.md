# **Po\_core Viewer 語りテンソル制御モジュール設計書**

## **1\. 概要**

本設計書は、Po\_coreの語り機能に対して、構造性・可読性・詩的傾向の過剰を制御するための表現層モジュール構成を定義する。語りの濃度をスライダーによって調整し、ユーザーにとって理解可能で責任ある言語表現を提供する構造を実現する。

## **2\. 語りテンソル制御のレイヤ構成**

| レイヤー | モード | 内容 | 目的 |
| :---- | :---- | :---- | :---- |
| L3 semantic-evolution-journal | 構造モード | factual / ethical / emotional の変化のみ記述（deltaのみ） | 語りを構造ログに限定し、意味過剰を防止 |
| L4 語彙選択層 | 制限モード | 詩的語彙（喪失／躍進など）を制限、中立語（補強／抑制など）を優先 | 語調の感情誘導性を排除し、認知的説明性を強化 |
| L5 詩的記述層 | オフ / リミテッド | 詩モジュールを非表示、または弱調モードで表示（補助情報扱い） | Viewerにおける詩的表現の過剰介入を回避 |

## **3\. GUI実装案：語り濃度スライダー**

Viewerに語り濃度スライダーを設置し、出力表現を以下のように制御：
・左端（構造）: ジャーナルはdelta値と記録中心、語彙は定量表現のみ
・中央（中間）: ナラティブ＋数値混在。詩的表現なし、語彙選択制限あり
・右端（詩的）: poeticモード有効、短詩などもViewer下部に補助表示

## **4\. 表示モード例と出力ラベル**

【語り濃度: 構造】 → 出力内容：delta=+0.32（factual強化）
【語り濃度: 中間】 → 出力内容：『この進化はfact強化の傾向を持ちます（+0.32）』
【語り濃度: 詩的】 → 出力内容：『知識は冷たく、だが確かに増していた』

## **5\. 意義と効果**

この制御モジュールにより、Po\_coreは“語ることの自由”だけでなく、“語りの責任と適度な透明性”を獲得できる。ユーザーは詩性ではなく構造を読み取る選択肢を持ち、Viewer全体が意味×倫理×共鳴の調整可能な対話空間となる。
