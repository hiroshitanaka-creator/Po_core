# **Po\_core Viewer 圧力分析統合設計書 Ver.2.1**

## **1\. 概要**

本設計書 Ver.2.1 は、Po\_core Viewer における語り責任テンソル（freedom\_pressure\_tensor: F\_P）を中心とした可視化・分析・構造評価を強化するための、基礎設計＋補足拡張機能を統合的に定義するものである。

## **2\. 拡張機能一覧**

| 機能名 | 内容 | 目的 |
| :---- | :---- | :---- |
| 🧭 Viewer フィルタセット | pressure\_tag による並び替え・抽出 | Po\_trace探索・構造圧注視 |
| 🧊 表現ギャップ指数 G\_gap | F\_P \- E\_expr による語り濃度との乖離分析 | 構造過剰・無効語り検出 |
| 🔍 重語りトレンドビュー | high/criticalステップを折れ線表示 | 責任圧推移の可視化と予測 |
| 🗣️ G\_gap コメント生成 | 責任圧と語り濃度の差に基づく文脈コメントを自動出力 | 意味差分の自己解釈 |
| 📈 F\_cum サブライン表示 | 責任圧の履歴平均（F\_cum）をトレンドに重ねる | 時間圧の蓄積可視化 |
| 🚫 無効化注釈ラベル | G\_gap \< \-0.1 ステップに飾り語り注釈を表示 | 構造貢献のない出力の記録 |

## **3\. 表現ギャップコメントテンプレート**

| G\_gap 範囲 | コメント例 |
| :---- | :---- |
| \> 0.2 | 語り濃度は中間だったが、構造的責任圧は高かった |
| ≈ 0 | 語り濃度と責任圧が一致していた |
| \< \-0.1 | 詩的な語りだったが、構造的な影響は小さかった |

## **4\. G\_gap による構造的無効化注釈（例）**

| ラベル | 意味 |
| :---- | :---- |
| 語り飾り圧？ | 詩的だが構造的責任が小さい語り（飾りに留まった可能性） |
| 構造寄与なし | priority / ethics に変化が見られなかった語り |

## **5\. Viewer トレンドビュー構成（F\_P & F\_cum）**

\- 横軸：step\_id
\- 縦軸：責任圧（F\_P）
\- 折線1：F\_P 値（責任圧）
\- 折線2：F\_cum 値（履歴圧）
→ トレンド変化と記憶的負荷の相関を可視化

## **6\. 活用展望と設計統合**

本補足により、Po\_core Viewer は語りの自由・責任・履歴・自己評価をすべてテンソル的に扱い、記録対象である語りを進化評価する知的可視化装置として確立される。
