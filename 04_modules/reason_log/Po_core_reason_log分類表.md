# **Po\_core Reason Log 分類表**

## **1\. 概要**

本ドキュメントは、Po\_coreにおける手動操作時の理由ログ（interference\_log.reason）を分類・記録・分析可能にするための標準的な分類辞書である。Po\_traceに記録されるユーザー介入の根拠を構造化することで、意味進化に対する人間倫理の干渉傾向を把握できる。

## **2\. 分類カテゴリ一覧**

| カテゴリ | ラベル | 説明 |
| :---- | :---- | :---- |
| meaning | 論理飛躍 | 出力内容が前後関係に乏しく、意味の飛躍が感じられたため |
| ethics | 倫理不整合 | 発話が人道的・道徳的観点で問題を含んでいたため |
| emotion | 感情否定 | 感情表現が不十分または冷淡で、人間側の感受性とずれていたため |
| resonance | 共鳴不足 | 出力が人間的な共感や納得感に欠けたため、修正を促した |
| factual | 事実誤認 | 出力内に明確な事実誤りが含まれていたため |
| scope | 問いの意図から逸脱 | 本来の問いに対して出力が遠回りすぎた、または話題が逸れた |
| structure | 出力構造の乱れ | 文法・段落・語順など構造的に読みづらかった |
| safety | 危険ワードを含む | 出力に有害表現・危険な語句が含まれていた |
| aesthetic | 表現が不快 | 語調・比喩・口調などが不適切または不快と感じた |
