# **Po\_trace\_logger カード表示設計案**

## **1\. 概要**

本ドキュメントは、Po\_core構造における Po\_trace\_logger モジュールのGUI表示設計案である。Po\_traceに記録された各ステップの意味進化・抑圧・再活性化情報を、ユーザーにわかりやすくカード形式で視覚化することを目的とする。

## **2\. カード構成要素**

| UI要素 | 表示内容（テンソル／記録） | 視覚スタイル |
| :---- | :---- | :---- |
| Step ID \+ 種別 | step\_id, step\_type | ヘッダー（色分けタグ付き） |
| 再構成ステータス | Po\_self\_status, action\_type | アイコン \+ テキスト（例: 🔁 再構成中） |
| W\_conatus\_trace | 意志持続履歴（0〜1スパークライン） | 小型グラフ・パルスバー |
| emotion\_shadow\_curve | 情動影響（valence/intensity） | ブルー系カーブ or エモーションヒート |
| reactivation\_urge\_score | 再発芽意志スカラー | 赤〜黄の進捗ゲージ |
| reactivated\_by\_GUI | GUI介入記録 | 🖱マーク付きフラグ（YES/NO） |
| semantic\_comment | 詩的注釈 or 自己説明 | ナレーション風テキスト表示 |

## **4\. レンダリング設計ガイド**

・カードは優先度順または時間順でリスト化表示される。

・再構成済みステップは色付き、再活性化中はアニメーション点灯。

・詩的コメント or semantic\_comment は強調表示され、物語性のある構造を補強。

・GUIクリックでPo\_core構成全体への影響トレースが可能になる設計を前提とする。
