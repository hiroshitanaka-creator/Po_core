# **Po\_trace強化GUI仕様書**

## **1\. 概要**

本仕様書は、Po\_coreシステムにおけるPo\_trace構造をGUI上で可視化・操作可能とするための強化案を記述したものである。特に、未発芽テンソルの再活性化を示唆する3つの新テンソル（W\_conatus\_trace、emotion\_shadow\_curve、reactivation\_urge\_score）をGUI表示上で明示化する。

## **2\. GUI表示要素定義**

| GUI要素 | テンソル | 表現方法 | 色・視覚的表現 |
| :---- | :---- | :---- | :---- |
| 再発芽可能性表示 | reactivation\_urge\_score | 透明度または赤み強度 | 赤：強 / グレー：弱 |
| 情動記憶カーブ | emotion\_shadow\_curve | カラーカーブ / ストライプ | 青系：抑制感情 / 黄系：希望感情 |
| 自己持続意志履歴 | W\_conatus\_trace | スパークラインまたはバーグラフ | 脈動表示（Pulseアニメ） |

## **3\. GUI構成例**

以下は、Po\_trace Viewer における出力ステップ表示の拡張例である。

***┌─────────────────────────────────────┐***  
***│ 🔁 Trace Step: reasoning\_1         │***  
***│ ● W\_conatus\_trace: 0.87           │***  
***│ ● Reactivation Urge: 0.84         │***  
***│ ● Shadow Emotion: (-0.4 / 0.7)    │***  
***│ 🔥 ヒートマップ色：深赤           │***  
***│ 🧭 コメント：「まだ語りたがっていた意味」 │***  
***└─────────────────────────────────────┘***

## **4\. 実装上の注意点**

・emotion\_shadow\_curveは時間経過に伴う記録のため、カーブ描画には時系列ベクトル処理が必要。

・W\_conatus\_traceとreactivation\_urge\_scoreは視覚的な優先表示制御に用いることが推奨される。

・Po\_self\_seedling と連動させることで、自己進化候補を視覚的に浮かび上がらせる。
