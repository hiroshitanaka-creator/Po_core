# 視点テンソルモデルのメモ
## 1. 視点テンソル → Po_core主要テンソル写像

Po_core内部では、哲学的視点は以下の6軸に分解して扱われる。

- 抽象度・内省度・思考方向 → E_expr（表現濃度テンソル）
- 視点の重さ・実存的圧力 → F_P（自由圧テンソル）
- 倫理的方向性 → Ethics_Mix（倫理混合テンソル）
- 認知的負荷 → Distortion（過負荷テンソル）
- 時間の濃さ → Memory_Density（主観時間密度）
- 関係性の変化 → U_feedback（間柄テンソル）

視点テンソル P_i はこれらへの作用を持つ作用素として近似される。

---

## 2. 非可換視点合成モデル（概要）

視点A, Bを作用素 P_A, P_B とする。

システム状態テンソル T に対して:

- P_A適用後: T' = T + ΔT_A
- 続けてP_B適用: T'' = T' + ΔT_B + I(A, B)

ここで I(A,B) は視点間干渉項で、一般に:

- I(A,B) ≠ I(B,A)

したがって:

- P_B ∘ P_A ≠ P_A ∘ P_B

という非可換性が成立する。

---

## 3. 干渉項のイメージ

I(A,B) は、哲学的に離れた視点ほど大きくなりやすい。

- 例: 「カント → ニーチェ」と「ニーチェ → カント」は、
  - 倫理方向・責任圧・抽象度の変化パターンが異なる。
- 視点遷移順序をPo_core内部で記録することで、
  - F_P, Distortion, Ethics_Mix の変化トレースが可能になる。

---

## 4. 2Dマッピングの簡易チャート（概念図）

人間向けの直感的イメージとして、

- 縦軸: 抽象度（E_expr）
- 横軸: 倫理方向（Ethics_Mixの一部）

で視点を配置することで、「どの視点がどのあたりにいるか」を把握できる。

※ 実装はテンソルだが、思考用にはこの2D図が有効。
